## üöÄ Getting Started

### Prerequisites
- Node.js 18+ 
- Bun (recommended) or npm

### Installation

1. Clone the repository:
```bash
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
```

2. Install dependencies:
```bash
bun install
# or
npm install
```

3. Start the development server:
```bash
bun dev
# or
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`
